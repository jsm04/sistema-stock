# Escenarios (ejemplos representativos)

**UC01 – Crear producto**

- Escenario básico:
  - Usuario ingresa nombre, SKU, categoría, precio y stock inicial
  - Sistema valida datos
  - Sistema guarda producto

- Escenario alternativo:
  - SKU duplicado → sistema rechaza operación

**UC03 – Registrar entrada de stock**

- Escenario básico:
  - Usuario selecciona producto
  - Ingresa cantidad > 0
  - Sistema actualiza stock
  - Sistema registra movimiento

**UC04 – Registrar salida de stock**

- Escenario básico:
  - Usuario selecciona producto
  - Ingresa cantidad
  - Sistema valida stock suficiente
  - Sistema descuenta stock

- Escenario alternativo:
  - Stock insuficiente → error

**UC05 – Alerta de stock mínimo**

- Escenario:
  - Stock ≤ umbral
  - Sistema marca producto como crítico
  - Sistema muestra alerta en UI

### 4. Tickets (formato tipo backlog ágil)

**TICKET 001 – Alta de producto**

- Descripción: Implementar creación de productos
- Criterios de aceptación:
  - Se puede ingresar nombre, SKU, categoría, precio, stock
  - SKU debe ser único
  - Datos obligatorios validados

**TICKET 002 – Listado de productos**

- Descripción: Mostrar inventario completo
- Criterios:
  - Lista paginada
  - Filtro por nombre y categoría

**TICKET 003 – Entrada de stock**

- Descripción: Registrar incrementos de inventario
- Criterios:
  - Cantidad > 0
  - Registro en historial

**TICKET 004 – Salida de stock**

- Descripción: Registrar decrementos de inventario
- Criterios:
  - No permitir stock negativo
  - Mensaje de error si no hay suficiente stock

**TICKET 005 – Historial de movimientos**

- Descripción: Visualizar trazabilidad
- Criterios:
  - Mostrar fecha, tipo, cantidad
  - Filtro por producto

**TICKET 006 – Alerta de stock mínimo**

- Descripción: Notificación de bajo stock
- Criterios:
  - Configurar umbral por producto
  - Indicador visible en UI

**TICKET 007 – Autenticación de usuarios**

- Descripción: Login básico
- Criterios:
  - Usuario/contraseña
  - Control de acceso

**TICKET 008 – Reporte de productos críticos**

- Descripción: Listar productos bajo mínimo
- Criterios:
  - Exportable (opcional)
  - Ordenado por nivel de stock
